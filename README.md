# JewlerySale

Vite and React jewelry showroom site with an interactive GLB necklace model.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Vercel

The project is configured for Vercel from the repository root:

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`

These settings are also defined in `vercel.json`.
