# JewlerySale

Lume Atelier 高级珠宝展示网站，包含系列宣传、私人预约入口与交互式 3D 项链鉴赏。

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Vercel

The project is configured for Vercel from the repository root:

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`

These settings are also defined in `vercel.json`.
